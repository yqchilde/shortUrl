Go+新浪抓到的接口生成短网址 -> t.cn

### Demo

[点我查看shortUrl](https://www.yqqy.top/tools/shorturl)

### Win Run

```shell
// download
git clone https://github.com/yqchilde/shortUrl.git

// start
go run main.go

// Browser running
http://127.0.0.1:8080/tools/shorturl
```

### Linux Run

`双击linux_build.bat`，交叉编译需要设置对应的 go env
